# 西电计算机通信网实验 2026

本仓库整理了西安电子科技大学计算机通信网实验相关资料，主要包括 Packet Tracer 拓扑文件、实验课件、配置截图、实验说明、实验大纲、报告封面以及安装辅助文件。

## 使用说明

1. `.pkt` 拓扑文件使用 Cisco Packet Tracer 打开，仓库内提供了 Packet Tracer 5.3 安装包和汉化包。
2. 实验资料仅供学习、复习和报告整理参考，请结合自己的实验环境、IP 地址规划和课程要求修改。
3. 各实验目录下的 `assets` 文件夹保存了配置过程、验证结果和拓扑截图。
4. 每个实验目录提供 `README.md`，用于快速查看拓扑、关键配置点和验收现象；实验 8 当前为预留目录。
5. 建议先阅读 [实验总览](<实验/README.md>)，再按实验编号进入对应目录打开 `.pkt` 和分实验说明。

## 推荐阅读顺序

1. 安装 Packet Tracer，确认 `.pkt` 文件可以正常打开。
2. 阅读 [实验总览](<实验/README.md>)，了解每个实验的主题和通用检查顺序。
3. 按 `实验/编号/README.md` 完成单个实验的拓扑、配置、验证和截图整理。
4. 考前或写报告前阅读 [计网考试宝典](<实验/计网考试宝典.md>)，按排错顺序复查 VLAN、Trunk、路由、DHCP、NAT 和 ACL。

## 说明

本仓库内容用于课程实验整理与复习参考。如需补充后续实验文档，建议按照 `实验/编号/` 的结构存放：

- `.pkt`：Packet Tracer 拓扑文件
- `assets/`：实验截图、配置过程和验证结果
- `README.md`：该实验的拓扑说明、关键配置点、验证现象和排错清单
- 课件文件：教师课件或实验要求原始资料
