# 西电计算机通信网实验 2026

本仓库整理了西安电子科技大学计算机通信网实验相关资料，主要包括 Packet Tracer 实验文件、实验课件、截图素材、实验大纲、报告封面以及安装辅助文件。

## 使用说明

1. 实验拓扑文件使用 Cisco Packet Tracer 打开，仓库内提供了 Packet Tracer 5.3 安装包和汉化包。
2. 实验文件仅供学习参考，请结合自己的实验环境、IP 地址规划和报告要求修改，不建议直接照抄。
3. 图片素材位于各实验目录下的 `assets` 文件夹，用于记录配置过程和实验结果。
